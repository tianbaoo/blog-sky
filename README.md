# 博客
线上地址：[博客天地](http://207.246.124.116/) 请挂梯子访问,否则速度会很慢。
# 开发环境
Django==1.10.6
Python==3.6.1
Pillow==4.2.1
Boostrap
# 功能
1. 写博客 （kindeditor-4.1.10 编译器）
2. Form组件验证用户登录和注册
3. 用户头像上传和预览
4. 登陆后后台可添加个人分类、标签
5. 登陆后可点赞、踩、评论他人博客
6. 进入个人后台可组合搜索个人文章
7. 登录和注册时动态生成图片验证码
8. 分页功能
9. 对用户发的文章和评论内容进行xss攻击检查，过滤关键词、script标签
